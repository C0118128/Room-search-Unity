# 部屋探索ゲーム ver.1.1 ~ Room-search-Unity ~

## 概要
小さな家の中を探索するゲーム。</br>
Unityの使い方を知るために作成した。今後、機能を追加する予定。

## 機能&技術
ver.1.0
* 家の構築(無料Asset)
* 家具配置(無料Asset)
* 視点移動(Unity object/script)
ver.1.1
* 番号型のアイテム(Unity object/script)
* 画面上部にアイテム欄実装(Unity object/script)

## バージョン詳細
* ver.1.0: 各階の機能実装
* ver.1.1: 1階、右側の棚に番号型のアイテムを配置 & 画面上部にアイテム欄実装
